注释

有多种注释方法，这里就先记录常用到的即可（记那么多也没用…）

使用 rem 命令


行间注释

多行注释
这里涉及到了goto命令，用法和C相同。标识符的有效长度为8个字符（如果小于8个字符，则识别前8个字符），可以带空格。

命令

所有命令可用参数/?查看帮助，以下只介绍最常用的。

echo

开启/关闭命令的回显

@

@的作用是禁止当前的命令回显再屏幕上，可以加在rem、echo等命令前面（注意：其优先级高于echo on，也就是说即使开启命令回显echo on，加@前的命令也不会回显。）

call

调用另一个批处理文件


调用标号后的命令(调用后会返回)

set

用于设置变量为指定字符串，在使用变量名时，要用%来表示。（另外这里需要注意，bat脚本语言中的等号左右不要有空格，否则容易出现BUG？）


/a表示变量名为数字表达式

start

重新启动一个单独的命令行窗口，在新窗口种执行指定的命令。在以下命令中，"hello"表示新窗口的标题名，/wait表示等待新窗口结束再执行下一条命令，type a1.txt表示传入新窗口后所要执行的命令。

if

判断是否存在a1.txt文件。注意if...else要写在同一行


如果不想写成同一行，可以如下分行（貌似(有连接下一行的效果）

字符串比较可用/i参数

for

循环显示当前目录下txt文件的内容

setlocal

设置该程序自身的环境变量，且不会影响到系统的环境变量。与之配套使用的是endlocal，而设置的环境变量只在这setlocal和endlocal之间有效。

shift

\n后的值只能是0~8
