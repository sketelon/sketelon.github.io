驱动双机调试

环境：VS2019

1. 开启调试功能

在被调试机输入命令：

bcdedit /debug on

bcdedit /dbgsettings net hostip:192.168.232.132 port:55555 key:bcn.own.safe.key

bcdedit /dbgsettings

hostip 指的是装有 vs2019 的机器的 ip 地址

2.安装必要程序

在装有 vs2019 的机器中搜索 WDK Test Target Setup，安装在被调试机中。

3. 配置 vs2019

Network host name 指的是被调试机的 ip 地址

Host IP 指的是本机 ip 地址

Windows 7 驱动调试 DbgView 不显示问题

Windows Registry Editor Version 5.00
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Debug Print Filter]
"DEFAULT"=dword:0000000f
