Github 命令

生成 ssh 密钥

1. 配置账户信息

git config --global user.name "你的GitHub用户名"

git config --global user.email "你的GitHub注册邮箱"

2.生成密钥

登录 Github，进入 Settings—>SSH and GPG keys—>New SSH key，将生成的 id_rsa.pub 文件中的内容拷贝到 key 中

ssh-keygen -t rsa -C "你的GitHub注册邮箱"

检测 GitHub 公钥设置是否成功

ssh git@github.com

本地提交到仓库

提交到缓冲区: git add ./

提交到仓库：git commit -m "comment"

从本地同步到仓库：git push

从仓库同步到本地：git pull

关于切换用户后VMTOOLS失效
